It is powered by a front-end builder tool for ease of use. === Parameters === '''title''' - The title of the timeline. (Optional) '''start_date''' - The start date of the timeline. Can be a number (e.g., -200) or a date string (e.g., 1995-01-01). (Required) '''end_date''' - The end date of the timeline. Can be a number (e.g., 500) or a date string (e.g., 2024-12-31). (Required) '''minor_increment''' - The numerical increment for the small tick marks. (Optional) '''major_increment''' - The numerical increment for the large tick marks. (Optional) '''1, 2, 3...''' - Each event is a separate numbered parameter containing a JSON object. Each object must contain date, label, and optionally description fields. '''Note:''' The timeline builder tool handles the JSON formatting for you. === Recommended Workflow === Open the Timeline Builder Tool file in your web browser. Fill out the form with your timeline data. The live preview will update automatically. Copy the generated wikitext from the "Generated Wikitext" text area. Paste the copied code directly onto a page on your wiki. The timeline will render correctly. === Example === The following code will generate a sample timeline. 
 {{Timeline |title=Major Life Events |start_date=1990 |end_date=2030 |minor_increment=5 |major_increment=10 |1={ "date": "1995", "label": "Born", "description": "Sarah was born." } |2={ "date": "2013", "label": "Graduated", "description": "High school graduation." } |3={ "date": "2020", "label": "First Book", "description": "Published her first book." } }}
